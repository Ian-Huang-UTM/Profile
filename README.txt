Directory Structure
-------------------
- **css\**: Contains Stylesheets CSS files for the template.
- **images\**: Holds Images used in the template.
- **scripts\**: Stores JavaScript used in the template.
- **styles\**: Exclusive to Purchased versions, it includes SCSS source files for stylesheets CSS.
- **favicon.ico**: Placeholder for the Favicon in the template.
- **index.html**: The main HTML page to open the template in the browser.
- ***.html**: Additional HTML pages (if any).
- **LICENSE-*.txt**: License file based on the downloaded variant.

Editing the Template
------------------------

1. **Editing Content in the Template**

   To modify the content, open the corresponding *.html files using an editor like VSCode or Notepad++.
   Search for the content you wish to change and edit it to the desired text.
   Save the file and reopen the HTML file in the browser to view the changes.

2. **Editing Styles of the Template**

   The main style is located in css\main.css, which you can edit if you understand CSS files.
   For premium versions, SCSS source files are provided in styles\, generating all template styles.
   Use these SCSS files to make changes and regenerate the CSS.

3. **Changing Images**

   Replace images within the images\ folder, ensuring the new file shares the original file's name.
   If renaming the image file, update the corresponding reference in the HTML file.

Using the Template
-----------------------
1. Upload the contents to your webserver.
2. Ensure index.html is in the main (root) folder of your webserver.
3. Avoid renaming template folders or files unless you are knowledgeable and update the names in HTML files accordingly.
4. For web hosting, consider using https://render.com/.
